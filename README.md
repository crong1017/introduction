### 1. 為什麼想要學習使用 GitHub
隨著學習的進步，經常需要使用多種軟體來撰寫程式碼，這不僅轉換麻煩，還不利於與同學協同編輯。GitHub 作為一個熱門的版本控制平台，不僅能有效管理程式碼版本，還讓我們可以即時進行協作。在這次活動中，我有機會深入學習 GitHub，並從中收穫了許多新知識，大大提升了編程效率和協作能力。

### 2. 平常愛好
我喜歡唱歌、手作以及服務他人，因此在校內積極參與各類服務性質的社團活動，像是醒新愛愛社。這些活動不僅讓我能夠奉獻自我，也讓我體驗到服務的意義。
也基於對科技應用的興趣，我參加了許多數位學伴計畫，希望能結合數位科技與服務，將學到的知識分享給更多人，尤其是偏鄉學童，讓他們也能接觸到科技的便利。

### 3. 遠程目標
我的長期目標是精通多種程式語言，並深入學習大數據分析、AI 應用等技術。我希望將這些技術與人機互動結合，設計並開發可供教學的行動裝置或學習機器人。透過這些產品，不僅可以輔助學習，更能將資訊科技融入日常教育中!
